Enum

Enums allows us to define a set of named constants.
Typescript provides two types of enums
1. Numeric Enums

						enum Direction {{'{'}}
						Up = 1,
						Down,
						Left,
						Right,}
					
						
							enum  E {{'{'}}
							A = getSomeValue(),
							B,   // error! 'A' is not constant-initialized,
							so 'B' needs an initializer}
2. String Enums

							enum Direction {{'{'}}
							 Up = "UP",
							Down = "DOWN",
							Left = "LEFT",
							Right = "RIGHT" },

There are also Heterogenous Enums: both numeric and string enums mixed
								
									enum Heteroenum {{'{'}}
									No = 0,
									Yes = "YES"}

Key factors of Enum:
1. Constant and Computed Members

										enum FileAccess {{'{'}}
										// constant members
										None,
										Read    = 1 << 1,
										Write   =1 << 2,
										ReadWrite  = Read | Write,
										// computed member
										G = "1213".length}
2. Enums are real objects that exist at runtime

4. const Enums
